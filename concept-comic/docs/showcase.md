# Showcase

This file contains quick examples for demonstrating the skill.

## Opportunity Cost

Input:

```text
机会成本是什么意思？
```

Output direction:

```text
A person chooses one door while the best unchosen door fades.
```

Labels:

```text
选择 / 放弃 / 最好替代 / 机会成本
```

Takeaway:

```text
选了一个，也放弃了另一个最好的可能。
```

## Inflation

Input:

```text
通货膨胀怎么理解？
```

Output direction:

```text
Money tickets chase limited goods; general price tags rise; the same wallet buys fewer goods.
```

Labels:

```text
钱 / 商品 / 价格上涨 / 购买力下降
```

Takeaway:

```text
不是一种东西变贵，而是总体价格持续上涨。
```
