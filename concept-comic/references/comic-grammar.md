# Comic Grammar

## Core rules

1. One comic image explains one core expression point.
2. One panel advances one step inside that expression point.
3. Characters must perform meaningful actions.
4. Reading order must be obvious.
5. Text must be short.
6. The comic should include conflict, choice, obstacle, change, or discovery.
7. End with a plain-language takeaway.
8. Avoid becoming a PPT infographic.

## Common story rhythms

### Problem → Mechanism → Result

Good for process concepts.

```text
Question appears → system works → key mechanism happens → result appears
```

### Misconception → Correction → Better model

Good for hard concepts.

```text
Wrong idea → why it fails → better metaphor → takeaway
```

### Choice → Sacrifice → Real cost

Good for trade-offs.

```text
Options appear → one is chosen → best alternative fades → takeaway
```

### Cause → Chain → Consequence

Good for causal concepts.

```text
First trigger → intermediate steps → final effect → takeaway
```

## Text rules

Recommended limits:

- Title: under 12 Chinese characters when possible
- Label: 2–6 Chinese characters
- Caption: one short sentence
- Takeaway: one sentence

Use labels like:

```text
选择
放弃
证据
检索
环境压力
购买力下降
```

Avoid paragraphs inside the image.

## Action verbs

Use visible actions:

```text
choose
search
carry
connect
compare
repair
block
pass
transform
discover
balance
chase
hide
open
close
fade
```

## Readability

The viewer should understand the main idea even if they only read:

```text
title + labels + final takeaway
```
