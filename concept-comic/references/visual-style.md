# Visual Style

## Positioning

This skill uses a clean, modern, educational comic style.

It should look like:

```text
a concept explanation from a smart science comic
```

Not like:

```text
a corporate presentation slide
```

## Visual requirements

- 16:9 horizontal image
- clear comic layout
- light background
- simple but expressive characters
- clear linework
- moderate colors
- strong visual hierarchy
- readable Chinese labels
- enough whitespace
- one obvious main action

## Character style

Characters can vary by concept:

- student
- teacher
- scientist
- detective
- explorer
- shopkeeper
- policymaker
- little robot
- data packet
- animal characters
- abstract concept characters

Do not use or copy another project’s fixed IP.

## Color

Use 2–4 main colors.

Use color to show:

- key object
- cause-effect relationship
- contrast
- danger / warning
- selected path
- fading alternatives

Avoid random colorful clutter.

## Text

Chinese text must be:

- short
- large enough
- readable
- directly tied to the visual action

## Avoid

- dense infographic
- tiny labels
- realistic poster style
- dark sci-fi style by default
- text-heavy educational slide
- decorative art with no explanatory structure
- copying a known project’s unique visual signature
