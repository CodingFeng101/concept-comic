# Concept Understanding

Before creating any comic, build a Concept Card.

## Concept Card

```text
Concept name:
Domain:
Target audience:
One-sentence explanation:
Core question:
Key components:
Key relationship:
Process / causal chain:
Common misconception:
Must-preserve knowledge:
Metaphor risk:
```

## Explanation ladder

When explaining a concept, use a three-layer ladder.

### Layer 1: Beginner sentence

One plain sentence.

Example:

```text
机会成本是你选择一个选项时，放弃的那个最好选择。
```

### Layer 2: Structure

Name the important parts.

Example:

```text
选择者、多个选项、被选择的选项、被放弃的最好选项。
```

### Layer 3: Boundary

Clarify what the concept is not.

Example:

```text
机会成本不是所有放弃选项的总和，而是最好的那个被放弃选项。
```

## Must-preserve knowledge examples

- Opportunity cost: best foregone alternative, not all alternatives combined.
- Entropy: tendency toward more probable/disordered states, not simply “trash.”
- Natural selection: no intentional decision to evolve.
- Inflation: sustained rise in general price level, not one product becoming expensive.
- RAG: retrieval plus generation, not model fine-tuning.
- Black hole: escape velocity exceeds light speed near the event horizon; not a cosmic vacuum cleaner.
- Prisoner’s dilemma: incentive structure, not a moral fable.
- Path dependence: early choices constrain later options, not absolute destiny.
