# Anti-patterns

Avoid these common failures.

## 1. Decorative-only image

Bad:

```text
A beautiful picture loosely related to the concept.
```

Why bad:

```text
It does not explain.
```

Fix:

```text
Add visual action that represents the concept relationship.
```

## 2. PPT pretending to be comic

Bad:

```text
Many boxes, arrows, and paragraphs.
```

Fix:

```text
Use characters, actions, panels, and short labels.
```

## 3. Wrong metaphor

Bad:

```text
Natural selection as animals deciding to upgrade themselves.
```

Fix:

```text
Show variation, environment pressure, survival, and reproduction.
```

## 4. Too many ideas

Bad:

```text
One image explains definition, history, mechanism, equations, applications, and criticism.
```

Fix:

```text
Pick one expression point. If that point still cannot stay simple in one image, split it into a series.
```

## 5. False certainty

Bad:

```text
Risk means bad things will happen.
```

Fix:

```text
Risk means uncertainty about outcomes.
```

## 6. Moralizing neutral concepts

Bad:

```text
Inflation happens because bad shopkeepers are greedy.
```

Fix:

```text
Show general price level, demand, supply, money, and purchasing power.
```

## 7. Text wall

Bad:

```text
Long definitions inside the image.
```

Fix:

```text
Use labels and one takeaway.
```

## 8. Copying another visual IP

Bad:

```text
Use a nearly identical character, color palette, and layout from a known project.
```

Fix:

```text
Use a general educational comic style and flexible characters.
```
