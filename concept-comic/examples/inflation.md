# Example: Inflation

## Input

```text
通货膨胀怎么理解？
```

## Output

概念：通货膨胀

一句话解释：
通货膨胀是总体价格水平持续上升，使同样的钱能买到的东西变少。

概念类型：
Causal Concept

漫画隐喻：
很多钱票追逐有限货架上的商品，多个商品价签一起升高，钱包能买到的东西变少。

推荐版式：
因果链条型

分镜脚本：
1. 许多钱票冲向有限货架。
2. 多个商品价签同时升高。
3. 角色拿同样的钱，却只能买更少东西。
4. 总结：不是一种东西变贵，而是总体价格持续上涨。

图中文字：
标题：通胀是什么？
标签：钱、商品、价格上涨、购买力下降
总结：钱没变少，但能买的东西少了。

QA：
不要画成单个商家贪心涨价；通胀是总体价格水平变化。
