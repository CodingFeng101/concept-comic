# Example: Prisoner’s Dilemma

## Input

```text
囚徒困境是什么？
```

## Output

概念：囚徒困境

一句话解释：
囚徒困境说明两个个体各自选择看似最保险的策略时，可能得到对双方都更糟的结果。

概念类型：
Trade-off / Game Theory Concept

漫画隐喻：
两个角色在隔开的房间里，各自面对“合作”和“背叛”两个按钮，因为无法信任对方，最后都按下背叛。

推荐版式：
四格误区纠正型

分镜脚本：
1. 两人被墙隔开，不能沟通。
2. 每人面前都有“合作”和“背叛”按钮。
3. 两人都担心“如果他背叛，我合作就亏了”。
4. 两人都按背叛，结果板写着“双方都更糟”。

图中文字：
标题：囚徒困境
标签：合作、背叛、个人理性、集体更糟
总结：个人最优，不一定带来整体最优。

QA：
不要做成简单道德故事；重点是激励结构。
