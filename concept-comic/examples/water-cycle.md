# Example: Water Cycle

## Input

```text
水循环怎么画成漫画？
```

## Output

概念：水循环

一句话解释：
水循环是水在蒸发、凝结、降水和汇流之间不断循环的过程。

概念类型：
Cycle Concept

漫画隐喻：
水滴小角色从海洋出发，被太阳送上天空，变成云，再变成雨回到河流和海洋。

推荐版式：
循环路径型

分镜脚本：
1. 太阳加热海面，水滴升空，标签“蒸发”。
2. 水滴在天空抱团成云，标签“凝结”。
3. 云下雨，标签“降水”。
4. 水流回河流和海洋，箭头回到起点，标签“汇流”。

图中文字：
标题：水的一场旅行
标签：蒸发、凝结、降水、汇流
总结：水不是消失了，而是在不同地方循环。

QA：
保持循环箭头清楚，不要画成一次性流程。
