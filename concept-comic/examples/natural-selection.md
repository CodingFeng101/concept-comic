# Example: Natural Selection

## Input

```text
用漫画解释自然选择
```

## Output

概念：自然选择

一句话解释：
更适合环境的性状更容易帮助个体生存并传给下一代。

概念类型：
Process Concept

漫画隐喻：
森林里有不同颜色的甲虫，显眼甲虫更容易被鸟发现，伪装色甲虫更容易留下后代。

推荐版式：
四格流程型

分镜脚本：
1. 不同颜色甲虫出现，标签“差异”。
2. 鸟更容易发现亮色甲虫，标签“环境压力”。
3. 伪装色甲虫更容易躲过捕食，标签“更容易生存”。
4. 下一代伪装色甲虫比例变多，标签“性状传递”。

图中文字：
标题：自然选择怎么发生？
标签：差异、环境压力、生存、遗传
总结：不是想进化，而是适合环境的性状更容易留下来。

QA：
不要画成动物主动决定变强。
