# Example: Opportunity Cost

## Input

```text
把“机会成本”变成一张漫画解释图
```

## Output

概念：机会成本

一句话解释：
机会成本是你选择一个选项时，放弃的那个最好选择。

概念类型：
Trade-off / Choice Concept

漫画隐喻：
一个人站在三扇门前，打开其中一扇门，另外几扇门慢慢变淡。

推荐版式：
四格流程型

分镜脚本：
1. 角色站在三扇门前：工作、读研、创业。
2. 角色选择“工作”这扇门。
3. “读研”和“创业”门变淡，其中最亮的一扇标注“最好替代”。
4. 角色回头看，底部总结：成本不只是花掉的钱，也包括放弃的最好选择。

图中文字：
标题：机会成本是什么？
标签：选择、放弃、最好替代、机会成本
总结：选了一个，也放弃了另一个最好的可能。

QA：
不要把其他门画成爆炸或毁掉，因为机会成本不是永久失去所有可能。
