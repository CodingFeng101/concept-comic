# Example: Path Dependence

## Input

```text
路径依赖是什么意思？
```

## Output

概念：路径依赖

一句话解释：
路径依赖是早期选择会改变后续选择空间，让后来更容易沿着原来的路继续走。

概念类型：
Abstract / Causal Concept

漫画隐喻：
雪地里第一条脚印越来越深，后面的人即使看到别的路，也更容易沿着已有脚印走。

推荐版式：
单场景解释型

分镜脚本：
主画面：雪地上有一条很深的脚印路。
前方有几条新路，但已有路更清楚、更省力。
角色犹豫后继续走旧路。
底部总结：早期选择，会把后面的路越踩越深。

图中文字：
标题：路径依赖
标签：早期选择、旧路、更省力、选择空间
总结：走过的路，会影响后面怎么走。

QA：
不要画成“命中注定”；路径依赖是约束增强，不是完全没有选择。
