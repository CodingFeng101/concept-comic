# Example: Entropy

## Input

```text
用漫画解释熵增
```

## Output

概念：熵增

一句话解释：
熵增可以理解为系统更容易从有序走向无序，而保持有序需要额外能量。

概念类型：
Abstract / Process Concept

漫画隐喻：
一间整齐房间会慢慢变乱，想保持整齐就需要有人持续打扫。

推荐版式：
四格流程型

分镜脚本：
1. 整齐房间，标签“有序”。
2. 时间流逝，书本和衣服散落，标签“自然变乱”。
3. 角色拿着电池形状的“能量”努力打扫。
4. 房间恢复部分整齐，总结“乱起来容易，整齐需要能量”。

图中文字：
标题：熵增怎么理解？
标签：有序、无序、能量、维护
总结：乱起来容易，整齐需要能量。

QA：
房间变乱只是入门隐喻，不要把熵等同于垃圾。
