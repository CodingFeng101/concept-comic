# Example: Black Hole

## Input

```text
黑洞为什么连光都逃不出去？
```

## Output

概念：黑洞事件视界

一句话解释：
在黑洞事件视界内，逃离所需速度超过光速，所以连光也无法逃出。

概念类型：
Definition / Abstract Concept

漫画隐喻：
一个极深的引力井，光线小人爬到边缘但坡太陡，无法越过“事件视界”这条线。

推荐版式：
单场景解释型

分镜脚本：
主画面：一个深井形空间弯曲图，中间是黑洞。
一束光像小信使沿着斜坡向外跑，但在事件视界内无法越过边界。
边界标注“事件视界”。
底部总结：不是黑洞在吸光，而是里面没有能逃出去的路径。

图中文字：
标题：光为何逃不出？
标签：事件视界、光、逃逸速度、引力井
总结：越过这条线，逃出去需要超过光速。

QA：
不要把黑洞画成普通吸尘器；重点是事件视界和逃逸速度。
